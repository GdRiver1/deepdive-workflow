# Team Jellie Hub v4

Team Jellie Hub v4 is a Blogger-safe, local-first kitchen command dashboard for a landscape GE Kitchen Hub / Android browser screen.

This GitHub folder is the public source/reference location used by the Blogger draft launch page. The full working local source currently lives in Jason's AI tools workspace at:

```text
apps/team-jellie-hub-v4/
```

Local source files:

- `index.html`
- `styles.css`
- `app.js`
- `README.md`
- `STATUS.md`

Safety boundaries:

- No secrets, API keys, OAuth tokens, credentials, or private automation code belong in this dashboard.
- No live YouTube upload, Blogger publish, Govee control, Google Calendar write, email send, delete, or background agent action belongs here.
- The dashboard is localStorage-only and uses public/no-key weather references when enabled.

Blogger access note: the Blogger draft page is a launch/reference page for Hub v4, not the full interactive app runtime.
